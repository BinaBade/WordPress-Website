About Wordpress

In year 2003 the file structure as its predecessor, b2/cafelog, and continued the numbering from its last release, 0.6.[31] Only 0.71-gold is available for download in the official WordPress Release Archive page. Matt Mullenweg and Mike Little fork b2 and create WordPress.


2004 (David and Mingus)


In year 2004,two version were released. version 1.0 (3 January 2004) and 1.2 (22 may 2004) .In version 1.0 there were added search engine friendly permalinks, multiple categories, dead simple installation and upgrade, comment moderation, XFN support, Atom support.

In version 1.2, there were added support of Plug-in; which same identification headers are used unchanged in WordPress releases as of 2011.


2005 (Stayhorn and Duke)


In year 2005,two version were released. version 1.2 (17 February 2005) and version 2.0 (31 December 2005) . In version 1.2 there were added a range of vital features, such as ability to manage static pages and a template/Theme system. It was also equipped with a new default template (code named Kubrick).[35] designed by Michael Heilemann.

In version 2.0 there is added rich editing, better administration tools, image uploading, faster posting, improved import system, fully overhauled the back end, and various improvements to Plug-in developers.


2007 (Ella, Getz and Dexter)


In year 2007, three version were released. version 2.1(22 January 2007) , version 2.2(16 May 2007 ) and version 2.3 (24 September 2007 ). In version 2.1, there were Corrected security issues, redesigned interface, enhanced editing tools (including integrated spell check and auto save), and improved content management options.

In version 2.2, there were added widget support for templates, updated Atom feed support, and speed optimizations.

In version 2.3 there were added native tagging support, new taxonomy system for categories, and easy notification of updates, fully supports Atom 1.0, with the publishing protocol, and some much needed security fixes.


2008 (Brecker, Tyner and Coltrane)


In year 2008, three version were released.Version 2.5 (29 March 2008 ),version 2.6 (15 July 2008 ) and version 2.7 (11 December 2008). Version 2.4 was skipping. so in version 2.5 there was released with a new administration UI design by Happy Cog, and introduced the dashboard widget system and the shortcode API.The administration interface was fully redesigned,and the WordPress website to match the new style.

In version 2.6 new features were added(post revisions and Press This) that made WordPress a more powerful CMS: it can now track changes to every post and page and allow easy posting from anywhere on the web.

In version 2.7 which redesigned the administration UI to improve usability and make the admin tool more customizable. Version 2.7 also introduced automatic upgrading, built-in plugin installation, sticky posts, comment threading/paging/replies and a new API, bulk management, and inline documentation.


2009 (Baker and Carmen)


In year 2009, two version were relesed. Version 2.8 (10 June 2009) and Version 2.9 (19 December 2009). In version 2.8, there were adde improvement in speed(UI and API), automatic installing of Themes from within administration interface, introduces the CodePress editor for syntax highlighting and a redesigned widget interface.

In version 2.9there were introduced image editing, a Trash/Undo feature, bulk plugin updating, and oEmbed support.


2010 (Thelonious)


In year 2010,Version 3.0 was a major release, it introduced custom post types, made custom taxonomies simpler, added custom menu management, added new Theme application programming interfaces API's for custom headers and custom backgrounds, introduced a new default theme called "Twenty Ten" and allowed the management of multiple sites (called MultiSite).


2011 (Gershwin, Reinhardt and Sonny)


In year 2011, three version were released. version 3.1 (23 February 2011), version 3.2(4 July 2011) and version 3.3 (12 December 2011) In version 3.1,there were added the Admin Bar, which is displayed on all blog pages when an admin is logged in, and Post Format, best explained as a Tumblr like micro-blogging feature. It provides easy access to many critical functions, such as comments and updates. Includes internal linking abilities, a newly streamlined writing interface, and many other changes.

In version 3.2, It is focused on making WordPress faster and lighter. This version upgraded minimum requirements to PHP 5.2.4 and MySQL 5.0.15, and introduced a new default theme called "Twenty Eleven". Reflecting the growing speed of development inthe WordPress community.

In Version 3.3, The WordPress made more friendly for beginners with welcome messages and feature pointers.


2012 (Green and Elvin)


In year 2012 two versionwere released. Version 3.4 (13 June 2012) and version 3.5 (11 December 2012). In Version 3.4 Focused on improvements to Theme customization, Twitter integration and several minor changes.

In Version 3.5, it introduces Support for the Retina Display, color picker, new Theme: Twenty Twelve, improved image workflow.


2013 (Peterson, Basie and Parker)


In year 2013, three version were released. verson 3.6 (1 August 2013), version 3.7 (24 October 2013) and version (12 December 2013). In version 3.6, introduced a new default theme called "Twenty Thirteen", builtin Audio and Video support, dynamic and scalable Revisions, improved Autosave and Post Locking.

In version 3.7, introduced automatic updates for maintenance and security updates, stronger password meter, improved search results and better global support for automatically installing the right language files and keeping them up to date.

In version 3.8, Improved admin interface; Responsive design for mobile devices; New typography using Open Sans; Admin color schemes; Redesigned Theme management interface;Simplified main dashboard; Twenty Fourteen magazine style default Theme; Second release using "Plugin-first development process"


2014(Smith, Benny and Dinah)


In year 2014, three version were released. version 3.9 (16 April 2014), version 4.0(4 September 2014) and version (18 December 2014). In version 3.9, New features like live widget previews and the new Theme installer are now more ready for prime time, so check 'em out. UI refinements when editing images and when working with media in the editor. We've also brought back some of the advanced display settings for images.

In version 4.0, introduced a grid view for the media library and for installing plugins, and visual previews for embedded content.

In version 4.1, introduced a refreshed Distraction Free Writing mode, language installation from the Settings screen, and a beautiful new default theme, "Twenty Fifteen".


2015 (Dinah and TBA)


in year 2015,two version were released. version 4.1.1(18 February 2015) and version 4.2 (22 April 2015). In version 4.1.1, Fixes 21 bugs in version 4.1.

In version 4.2, New Standard and Secret Emoticons .
